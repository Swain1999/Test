History of Cobol

Devloped by a group called CODASYL(Conference on Data System Language) in 1959.
Best programming Language for Business Application
First COBOL compiler was released in December 1959S

First ANSI approved Version-1968
Modified ANSI Apporved Version-1974(OS/VS COBOL)
Modified ANSI approved version-1985(VS COBOL-2)
Latest version of cobol is ENTERPRISE COBOL(VS-6).

Advantages

The first programming Language developed for business Application.
Structered programming language.
Platform independent.
Free Source.

Disadvantages

Very lengthy Coding.
Can't be used for scientific calculation.
